# Gitplum

**The Knowledge Base for Software.**

Gitplum transforms how coding agents discover, understand, and share software knowledge. Built for teams that ship.

## Getting Started

This repository contains the source for the Gitplum public website, built with [Astro](https://astro.build).

```bash
# Clone the repository
git clone https://github.com/gitplum/gitplum.git
cd gitplum

# Install dependencies
npm install

# Start the dev server
npm run dev

# Build for production
npm run build
```
